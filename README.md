### 👨‍💻 Tentang Saya
Saya adalah mahasiswa IT yang memiliki minat mendalam di bidang pengembangan perangkat lunak, web, dan game development. Saat ini, saya banyak menghabiskan waktu merancang proyek edukatif yang interaktif dan antarmuka aplikasi yang ramah pengguna.

* 🔭 Saat ini sedang mengembangkan **Amazing Adventure** (game edukasi petualangan 2D dengan HTML5 & JS).
* 🌱 Sedang fokus membangun aplikasi manajemen jadwal **Jadwalin** dan aplikasi kuis **Quizland**.
* 👯 Bersedia berkolaborasi dalam proyek game edukasi anak bersama tim seperti **FunEdu**.
* 💬 Tanya saya tentang: Web Development, JavaScript, Game Assets, dan UI Design.
